## Technologies used:

* Java Script
* mongodb
* mogodb 
* mongodb atlas
* heroku
* express
* mongoose
* morgan


The application will be invoked by using the following commands:
```bash
npm i
```
```bash
npm run seed
```
```bash
npm start
```
